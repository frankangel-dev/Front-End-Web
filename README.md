# About California

A responsive front-end website highlighting the state of California and a few of its major cities. Built strictly with HTML, CSS, and JavaScript, it provides geographical, demographical, and historical information without relying on heavy front-end frameworks.

## Features

- **State Overview:** General information regarding California's landscape and economy.
- **City Profiles:** Dedicated pages for Sacramento, Los Angeles, and Anaheim featuring population statistics, average income levels, and top local attractions.
- **Responsive Design:** Adapts to different screen sizes for smooth desktop and mobile viewing.
- **Interactive Form:** A built-in contact form with client-side input verification.
- **Clean Layout:** A structured UI based on an HTML5 UP design, featuring easy-to-read content, sidebars, and clear navigation.
